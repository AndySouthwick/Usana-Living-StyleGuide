General Notes

Our official webfont is Montserrat for Latin and Latin Extended characters. Any USANA market that requires special language fonts will use their corresponding font in the Noto Sans family. Chinese will use Noto Sans CJK (Simplified or Traditional... must be specified and used).

Font Calculations

  • $font-size: 16px
  • $line-height: 1.35rem
  • $desktop-font-scale: 1.75rem
  • $mobile-font-scale: 1.4rem

Desktop

  • $h1-font-size-desktop = $font-size*($desktop-font-scale^3)
  • $h1-line-height-desktop = $h1-font-size-desktop*$line-height
  • $h2-font-size-desktop = $font-size*($desktop-font-scale^2)
  • $h2-line-height-desktop = $h2-font-size-desktop*$line-height
  • $h3-font-size-desktop = $font-size*($desktop-font-scale)
  • $h3-line-height-desktop = $h3-font-size-desktop*$line-height

Mobile

  • $h1-font-size-mobile = $font-size*($mobile-font-scale^3)
  • $h1-line-height-mobile = $h1-font-size-mobile*$line-height
  • $h2-font-size-mobile = $font-size*($mobile-font-scale^2)
  • $h2-line-height-mobile = $h2-font-size-mobile*$line-height
  • $h3-font-size-mobile = $font-size*($mobile-font-scale)
  • $h3-line-height-mobile = $h3-font-size-mobile*$line-height
